# Contributing to HireMirror AI

We love contributions! By participating, you help make HireMirror AI better for everyone.

## How to Contribute

1. **Fork the repository** and clone it locally:
   ```bash
   git clone https://github.com/ankitkumar04100/HireMirrorAI.git
   ```
2. **Create a new branch** for your feature or bug fix:
   ```bash
   git checkout -b feature/your-feature
   ```
3. **Make your changes** and ensure the code passes all tests.
4. **Commit your changes** with a clear message:
   ```bash
   git commit -m "Add new feature X"
   ```
5. **Push your branch** and open a pull request.

---

## Guidelines

- Follow the existing code style (Python: PEP8, JavaScript: Prettier/TailwindCSS)
- Write clear **commit messages**.
- Include **tests** for new features or fixes.
- Document any new APIs or UI changes in the README or docs folder.

---

## Reporting Bugs or Feature Requests

Please use the GitHub Issues tab to report bugs or request features. Include as much detail as possible, including screenshots or steps to reproduce.
